# CMPM 121 D3 Project

This is the starting point for your CMPM 121 D3 project. The goal of this project is to create a game that combines elements of 4096 and Threes with Pokemon Go. The project is an opportunity to practice the Flyweight, Facade, and Memento software design patterns.

This project was done in collaboration with Joseph Gonzalez. This is his Github:[GithubRepo](https://github.com/josephGonzalez1/)
